Dataset

The data from this project was taken from Kaggle's Toxic Comment Classification Challenge.

In this dataset, 159,571 Wikipedia comments were labeled by human raters for toxic behavior and split into the following non-exclusive categories:

  • toxic
  • severe_toxic
  • obscene
  • threat
  • insult
  • identity_hate


To be consistent with the definiton of "Hate Speech", a single cetegory was created from severe_toxic and identity_hate.
Comments that were only toxic, obscene, threat, or insult were treated as neutral, non-flagged speech.
